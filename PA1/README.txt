Summary of new methods:

smooth1D: linear interpolation of color (only used when doSmooth is true)

smoothTexture1D: linear interpolation of texture coordinate (only used when doTexture is true)

smoothTexture2D: bilinear interpolation of texture coordinate (only used when doTexture is true)

alpha: blend in the color of the point with the background (only used when doAA is true)

findLineBoundary: find the leftmost and rightmost (sub)pixel for every (sub)value of y (only used when drawTriangle)
